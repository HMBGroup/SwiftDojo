Binary Search Tree
==================

Implementation of a Binary Search Tree.

All operations have **O(h)** time complexity where **h** is the height of the tree.

Operations:
 - `search`
 - `insert`
 - `remove`


 Traversal methods:
 - Depth-first search:
  - Pre-order
  - In-order
  - Post-order
 - Breadth-first search
